# Quick Start

This guide will help you get started with Arnoldi by walking through a simple example of computing eigenvalues and eigenvectors of a sparse matrix.

## Basic Example

Here's a complete example that computes the 5 largest eigenvalues of a Markov walk matrix and compares the results with ARPACK:

```python
import numpy as np
from scipy.sparse.linalg import eigs
from arnoldi.krylov_schur import partial_schur
from arnoldi.matrices import mark
from arnoldi.utils import arg_largest_real

# Create a Markov walk matrix
A = mark(50)

# Configure parameters
TOL = 1e-8
MAX_RESTARTS = 1_000
MAX_DIMS = 20
K = 5
WHICH = "LR"
SORT_FUNCTION = arg_largest_real

# Compute partial Schur decomposition
Q, T, history = partial_schur(
    A,
    K,
    max_dim=MAX_DIMS,
    stopping_criterion=TOL,
    sort_function=SORT_FUNCTION,
    max_restarts=MAX_RESTARTS,
)

# Convert Schur transform into eigenpairs
vals, S = np.linalg.eig(T)
vecs = Q @ S

# Print results
print("========== Arnoldi Implementation =============")
for k in range(K):
    val, vec = vals[k], vecs[:, k]
    residual = np.linalg.norm(A @ vec - val * vec)
    norm_residual = residual / np.abs(val)
    print(
        f"Residual {k}, value is {np.real(val):.7}, "
        f"res is {residual:.3e}, norm res is {norm_residual:.3e}"
    )

# Compare to ARPACK
vals_arpack, vecs_arpack = eigs(
    A, K, ncv=MAX_DIMS, maxiter=MAX_RESTARTS, which=WHICH, tol=TOL
)

print("\n========== ARPACK (scipy) Implementation =============")
for k in range(K):
    val, vec = vals_arpack[k], vecs_arpack[:, k]
    residual = np.linalg.norm(A @ vec - val * vec)
    norm_residual = residual / np.abs(val)
    print(
        f"Residual {k}, value is {np.real(val):.7}, "
        f"res is {residual:.3e}, norm res is {norm_residual:.3e}"
    )
```

## Understanding the Code

Let's break down what this code does:

1. **Matrix Creation**: We create a Markov walk matrix using `mark(50)`, which is a 50x50 test matrix included in Arnoldi.

2. **Configure Parameters**:
   - `K = 5`: Number of eigenvalues to compute
   - `MAX_DIMS = 20`: Maximum dimension of the Krylov subspace before restart
   - `TOL = 1e-8`: Convergence tolerance
   - `MAX_RESTARTS = 1_000`: Maximum number of restarts allowed
   - `SORT_FUNCTION = arg_largest_real`: Function to sort eigenvalues (largest real part)

3. **Compute Partial Schur Decomposition**: The `partial_schur` function returns:
   - `Q`: Orthonormal basis for the Krylov subspace
   - `T`: Upper triangular Schur form (eigenvalues are on the diagonal)
   - `history`: Convergence history information

4. **Extract Eigenpairs**: Convert the Schur decomposition to eigenvalues and eigenvectors using `np.linalg.eig(T)` and `Q @ S`.

5. **Verify Results**: Compute residuals `||A*v - λ*v||` to verify the accuracy of the computed eigenpairs.

## Minimal Example

If you just want to compute eigenvalues with default settings:

```python
import numpy as np
from scipy.sparse import random
from arnoldi import partial_schur

# Create a random sparse matrix
A = random(1000, 1000, density=0.01, format='csr')

# Compute 6 eigenvalues with default settings
Q, T, history = partial_schur(A, nev=6)

# Extract eigenvalues
eigenvalues = np.linalg.eigvals(T)
print(f"Computed eigenvalues: {eigenvalues}")
```
