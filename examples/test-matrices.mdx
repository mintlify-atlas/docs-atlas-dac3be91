Arnoldi provides several built-in test matrix generators that are useful for testing, benchmarking, and understanding eigensolver behavior.

## Available Test Matrices

The `arnoldi.matrices` module provides two main test matrix generators:

1. **Markov matrices** - `mark(m)`: Random walk transition matrices
2. **Laplacian matrices** - `laplace(n)`: Symmetric tridiagonal matrices with known eigenvalues

## Markov Matrices: `mark(m)`

### Description

Generates a sparse Markov transition matrix for a random walk on a lower triangular grid with `m` rows.

```python
from arnoldi.matrices import mark

A = mark(50)
print(f"Matrix shape: {A.shape}")  # (1275, 1275) since n = m*(m+1)/2
print(f"Matrix type: {type(A)}")    # scipy.sparse.csr_matrix
```

### Characteristics

- **Size**: `n × n` where `n = m*(m+1)/2`
- **Type**: Sparse CSR matrix
- **Symmetry**: Generally non-symmetric
- **Eigenvalue properties**: 
  - Largest eigenvalue is typically close to 1
  - Well-suited for testing "LR" (largest real part) eigenvalue targeting
- **Use cases**: Testing non-symmetric eigensolvers, convergence behavior

### Example Usage

```python
import numpy as np
from arnoldi.krylov_schur import partial_schur
from arnoldi.matrices import mark
from arnoldi.utils import arg_largest_real

# Create a Markov matrix
A = mark(50)  # Creates a 1275 × 1275 matrix

# Compute eigenvalues
Q, T, history = partial_schur(
    A,
    k=5,
    max_dim=20,
    stopping_criterion=1e-8,
    sort_function=arg_largest_real,
    max_restarts=1000,
)

# Extract eigenvalues
vals, S = np.linalg.eig(T)
print("Largest eigenvalues:", vals[:5])
```

### Reference

See section 2.5.1 of *Numerical Methods for Large Eigenvalue Problems* (2nd edition) by Yousef Saad for more details and numerical examples.

## Laplacian Matrices: `laplace(n)`

### Description

Creates a 1D Laplacian operator of dimension `n`. This is a symmetric, tridiagonal matrix with `-2` on the main diagonal and `1` on the upper and lower diagonals.

```python
from arnoldi.matrices import laplace

A = laplace(100)
print(f"Matrix shape: {A.shape}")  # (100, 100)
print(f"Matrix type: {type(A)}")    # scipy.sparse.dia_matrix
```

### Characteristics

- **Size**: `n × n`
- **Type**: Sparse diagonal matrix (dia_matrix)
- **Symmetry**: Symmetric
- **Structure**: Tridiagonal with:
  - Main diagonal: `-2`
  - Upper/lower diagonals: `1`
- **Eigenvalues**: Known analytically (see below)

### Known Eigenvalues

The Laplacian matrix has analytically known eigenvalues, which makes it excellent for validation:

```python
from arnoldi.matrices import laplace, laplace_eigen
import numpy as np

n = 100
A = laplace(n)

# Get the true eigenvalues
true_eigenvalues = laplace_eigen(n)
print("True eigenvalues (first 5):", true_eigenvalues[:5])
```

The eigenvalues are given by the formula:

```
λ_k = -2 + 2*cos(k*π/(n+1))  for k = 1, 2, ..., n
```

### Example with Validation

```python
import numpy as np
from arnoldi.krylov_schur import partial_schur
from arnoldi.matrices import laplace, laplace_eigen
from arnoldi.utils import arg_largest_real

n = 100
A = laplace(n)

# Compute eigenvalues using Arnoldi
Q, T, history = partial_schur(
    A,
    k=6,
    max_dim=20,
    stopping_criterion=1e-10,
    sort_function=arg_largest_real,
    max_restarts=1000,
)

# Extract computed eigenvalues
vals_computed, S = np.linalg.eig(T)
vecs = Q @ S

# Get true eigenvalues
vals_true = laplace_eigen(n)

# Compare (Laplacian has real eigenvalues)
print("Computed eigenvalues (largest real part):")
for i in range(6):
    print(f"  {np.real(vals_computed[i]):.10f}")

print("\nTrue eigenvalues (largest):")
for i in range(6):
    print(f"  {vals_true[-(i+1)]:.10f}")

# Verify residuals
print("\nResiduals:")
for i in range(6):
    val, vec = vals_computed[i], vecs[:, i]
    residual = np.linalg.norm(A @ vec - val * vec)
    print(f"  Eigenvalue {i}: residual = {residual:.3e}")
```

### Expected Eigenvalue Properties

For a Laplacian of size `n`:

- **Largest eigenvalue**: `-2 + 2*cos(π/(n+1))` ≈ `-0.0002` for large `n`
- **Smallest eigenvalue**: `-2 + 2*cos(n*π/(n+1))` ≈ `-4` for large `n`
- **Spectrum**: All eigenvalues are real and lie in the interval `[-4, 0]`
- **Symmetry**: All eigenvectors are real and orthogonal

## Choosing a Test Matrix

### Use Markov matrices when:
- Testing non-symmetric eigensolvers
- Benchmarking "LR" (largest real) eigenvalue targeting
- Studying convergence behavior on non-Hermitian problems
- Comparing with examples from the literature (Saad's book)

### Use Laplacian matrices when:
- You need ground truth eigenvalues for validation
- Testing symmetric/Hermitian eigensolvers
- Verifying numerical accuracy
- Debugging eigensolver implementations

## Custom dtype Support

The `laplace` function supports custom data types:

```python
import numpy as np
from arnoldi.matrices import laplace

# Float32 for reduced memory
A_float32 = laplace(1000, dtype=np.float32)

# Complex128 for complex arithmetic
A_complex = laplace(1000, dtype=np.complex128)
```
