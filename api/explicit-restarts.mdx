## explicit_restarts_with_deflation

```python
explicit_restarts_with_deflation(
    A,
    nev,
    *,
    max_dim=None,
    stopping_criterion=None,
    max_restarts=100,
    sort_function=None
)
```

Compute eigenvalues using explicit restarts with deflation. This method deflates converged eigenvalues one at a time, building an orthonormal basis incrementally.

### Parameters

<ParamField path="A" type="ndarray of shape (n, n)" required>
  Square matrix to compute eigenvalues for.
</ParamField>

<ParamField path="nev" type="int" required>
  Number of eigenvalues to compute.
</ParamField>

<ParamField path="max_dim" type="int" default="min(max(2 * nev + 1, 20), n)">
  Maximum dimension of the Krylov subspace for each eigenvalue.
</ParamField>

<ParamField path="stopping_criterion" type="float" default="sqrt(eps)">
  Convergence tolerance for approximate residuals.
</ParamField>

<ParamField path="max_restarts" type="int" default="100">
  Maximum number of restarts per eigenvalue.
</ParamField>

<ParamField path="sort_function" type="callable" default="arg_largest_magnitude">
  Function to sort eigenvalues for selection.
</ParamField>

### Returns

<ResponseField name="eigenvalues" type="ndarray of shape (nev,)">
  Computed eigenvalues.
</ResponseField>

<ResponseField name="eigenvectors" type="ndarray of shape (n, nev)">
  Computed eigenvectors (one per column).
</ResponseField>

<ResponseField name="history" type="History">
  Convergence history with matvec counts and restart counts per eigenvalue.
</ResponseField>

### Example

```python
import numpy as np
from arnoldi.explicit_restarts import explicit_restarts_with_deflation
from arnoldi.matrices import laplace
from arnoldi.utils import arg_largest_real

# Create a test matrix
A = laplace(100).toarray()

# Compute 5 eigenvalues
vals, vecs, history = explicit_restarts_with_deflation(
    A,
    nev=5,
    max_dim=20,
    stopping_criterion=1e-8,
    sort_function=arg_largest_real
)

print("Eigenvalues:", vals)
print("Matvecs per eigenvalue:", history.matvecs)
print("Restarts per eigenvalue:", history.restarts)
```

### Notes

This method is generally less efficient than Krylov-Schur but can be useful for comparison and understanding. It computes eigenvalues sequentially with deflation using Modified Gram-Schmidt to maintain orthogonality against previously converged eigenvectors.

## naive_explicit_restarts

```python
naive_explicit_restarts(
    A,
    m=None,
    *,
    stopping_criterion=None,
    max_restarts=10
)
```

Compute a single eigenvalue using naive explicit restarts without deflation. This is primarily for educational purposes and comparison.

### Parameters

<ParamField path="A" type="ndarray of shape (n, n)" required>
  Square matrix to compute eigenvalue for.
</ParamField>

<ParamField path="m" type="int" default="min(max(3, 20), n)">
  Dimension of the Krylov subspace.
</ParamField>

<ParamField path="stopping_criterion" type="float" default="sqrt(eps)">
  Convergence tolerance.
</ParamField>

<ParamField path="max_restarts" type="int" default="10">
  Maximum number of restarts.
</ParamField>

### Returns

<ResponseField name="ritz" type="RitzDecomposition">
  RitzDecomposition containing the computed eigenvalue and eigenvector (only k=1 supported).
</ResponseField>

<ResponseField name="converged" type="bool">
  True if the method converged within max_restarts.
</ResponseField>

<ResponseField name="n_restarts" type="int">
  Actual number of restarts performed.
</ResponseField>

### Example

```python
from arnoldi.explicit_restarts import naive_explicit_restarts

A = np.random.randn(50, 50)
ritz, converged, n_restarts = naive_explicit_restarts(A, m=15, max_restarts=20)

if converged:
    print(f"Converged in {n_restarts} restarts")
    print(f"Dominant eigenvalue: {ritz.values[0]}")
else:
    print("Did not converge")
```

### Notes

This implementation only supports computing a single eigenvalue (k=1) and is not recommended for production use. Use `partial_schur` or `explicit_restarts_with_deflation` instead.

## History

```python
@dataclass
class History:
    matvecs: np.ndarray
    restarts: np.ndarray
```

Dataclass for tracking convergence history of eigenvalue computations.

### Attributes

<ResponseField name="matvecs" type="ndarray of shape (k,), dtype=int32">
  Number of matrix-vector products required to converge each eigenvalue.
</ResponseField>

<ResponseField name="restarts" type="ndarray of shape (k,), dtype=int32">
  Number of restarts required to converge each eigenvalue.
</ResponseField>

### Properties

#### k

```python
@property
k(self) -> int
```

Number of eigenvalues tracked (length of matvecs/restarts arrays).

#### total_matvecs

```python
@property
total_matvecs(self) -> int
```

Total number of matrix-vector products across all eigenvalues.

### Class Methods

#### from_k

```python
@classmethod
from_k(cls, k: int) -> History
```

Create a History object initialized with zeros for k eigenvalues.

### Example

```python
from arnoldi.explicit_restarts import History

# Create history for 5 eigenvalues
history = History.from_k(5)

# After computation, access results
print(f"Total matvecs: {history.total_matvecs}")
print(f"Average restarts: {history.restarts.mean():.1f}")
print(f"Most expensive eigenvalue: {history.matvecs.argmax()}")

for i in range(history.k):
    print(f"Eigenvalue {i}: {history.matvecs[i]} matvecs, {history.restarts[i]} restarts")
```
